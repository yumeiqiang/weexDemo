
### weex Demo
#### dist ---打包之后的jsbudles文件

### web运行
``` js
 yarn install（npm install）
npm run build --编译
 npm run --watch  --监听编译
 npm run serve   --开启localhost 服务
```


## iOS 运行
``` js
 weex platforms add ios
 npm run copy:ios
 进入iOS目录（默认已安装pod） pod install  cmd+r编译
```




### android
```
 weex platforms add android
 npm run copy:android
 android studio 编译
```
